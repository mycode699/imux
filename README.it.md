# imux

`imux` e un centro di comando nativo per macOS pensato per il lavoro con l'IA. Riunisce rendering del terminale di livello Ghostty, explorer locali e remoti, visualizzazione e modifica dei file dentro l'app e un livello di supervisione per pianificazione e avanzamento.

Repository: <https://github.com/mycode699/imux>

Docs: [English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md)

## Che cos'e `imux`

- uno spazio di lavoro terminale nativo per macOS basato su `libghostty`
- explorer locale e remoto via SSH con apertura, modifica e salvataggio dei file
- supervisore e automazione LLM per avvio rapido e follow-up operativo
