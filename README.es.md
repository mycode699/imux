# imux

`imux` es un centro de mando nativo para macOS orientado al trabajo con IA. Reune renderizado de terminal al nivel de Ghostty, exploradores locales y remotos, vista y edicion de archivos dentro de la app y una capa supervisora para planificar y seguir la ejecucion.

Repository: <https://github.com/mycode699/imux>

Docs: [English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md)

## Que es `imux`

- un espacio de trabajo de terminal nativo para macOS basado en `libghostty`
- explorador local y remoto por SSH con abrir, editar y guardar archivos
- supervisor y automatizacion con LLM para arrancar rapido y dar seguimiento al trabajo
