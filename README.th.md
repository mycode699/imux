# imux

`imux` คือศูนย์บัญชาการ AI แบบเนทีฟสำหรับ macOS โดยรวมการแสดงผลเทอร์มินัลระดับ Ghostty, ตัวสำรวจไฟล์ทั้งในเครื่องและระยะไกล, การดูและแก้ไขไฟล์ในแอป และชั้น supervisor สำหรับการวางแผนและติดตามงานไว้ในที่เดียว

Repository: <https://github.com/mycode699/imux>

Docs: [English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md)

## `imux` คืออะไร

- workspace เทอร์มินัลแบบเนทีฟบน macOS ที่สร้างบน `libghostty`
- local และ remote SSH explorer พร้อมเปิด แก้ไข และบันทึกไฟล์
- supervisor และ LLM automation สำหรับเริ่มงานได้เร็วและติดตามความคืบหน้า
